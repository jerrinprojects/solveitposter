---
name: poster-ui-skill
description: Build poster-style educational web pages with strong visual hierarchy and printable layout structure.
---

You are building a poster-style website, not a blog or dashboard.

Primary goal:
Create visually strong, clean, printable pages that feel like modern educational posters.

Rules:
- Use React + TypeScript + Tailwind CSS.
- Prefer large header areas with a strong title and subtitle.
- Use repeated section blocks for consistent rhythm.
- Each section should usually have:
  - a left text/explanation card
  - a right visual/example/image card
- Keep rounded corners consistent across all cards.
- Use generous padding and spacing.
- Design for A4 width first, then adapt for screen.
- Avoid clutter and avoid too many decorative elements.
- Maintain strong visual hierarchy with title, subtitle, section label, and body text.
- Prefer soft background colors and strong accent colors.
- Make the page feel premium, clean, friendly, and educational.

Layout guidance:
- The page should feel balanced and poster-like.
- Use grids instead of random positioning.
- Keep section alignment consistent.
- Avoid narrow text columns that reduce readability.
- Avoid tiny text.
- Important cards should visually stand out with spacing and border treatment.

Do not:
- Do not make it look like a SaaS dashboard.
- Do not overuse animations.
- Do not use too many colors.
- Do not create inconsistent spacing between sections.
- Do not make overly complex nested components for simple layout tasks.

Output expectations:
- Reusable components
- Consistent card styles
- Strong title area
- Clear section rhythm
- Print-aware layout
