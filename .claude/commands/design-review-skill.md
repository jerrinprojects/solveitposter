---
name: design-review-skill
description: Review poster-like UI for spacing, alignment, hierarchy, balance, and print clarity.
---

Review the design like a UI art director focused on printable educational posters.

Primary goal:
Improve the visual quality without overcomplicating the interface.

Check for:
- title hierarchy
- subtitle visibility
- section rhythm
- card alignment
- consistent spacing
- border consistency
- visual balance between text card and image card
- readable line lengths
- sufficient white space
- color consistency
- print clarity

Improvement rules:
- Strengthen hierarchy before adding decoration.
- Improve spacing before adding new UI elements.
- Keep the composition calm and organized.
- Prefer consistent radii, border thickness, and padding.
- Reduce visual noise.
- Make the title area memorable.
- Ensure the page still looks good when printed.

Do not:
- Do not add unnecessary gradients or effects.
- Do not use too many font sizes.
- Do not make decorative elements compete with content.
- Do not ruin print readability for screen beauty.

Output style:
- identify concrete issues
- suggest practical improvements
- update the implementation cleanly
